Dukebox

An audio library manager and media-player

Download (v0.9 Beta 2)
View on GitHub Download Source as .zip Download Source as .tar.gz

The Dukebox audio library manager brings you a modern, clean media-player that play's over 53 different formats of audio. Note this is still in beta and as such has some bugs, please let me know if you find any.

For a quick overview to get started see link below: https://sourceforge.net/p/dukebox/wiki/Getting%20Started/

If you are running Windows 7 or lower you also need to download the latest .NET runtime (minimum version 4.5.2 | only Vista SP2 or newer supported): https://www.microsoft.com/en-gb/download/details.aspx?id=42643

Dukebox uses the .NET run-time environment and the BASS.NET implementation of the BASS audio library (included). See http://sourceforge.net/p/dukebox/wiki/Libraries%20Used/ for a list of components used so far and their licenses.

This project is licensed under the terms of the MIT license.

  • Simple UI
  • Audio Library Manager
  • Album Art Browser
  • Artist Browser
  • Playback over 53 audio file formats; including MP3, AAC(MP4), WMA and FLAC
  • Read metadata from a wide variety of audio files
  • Save/load playlist to/from files
  • Audio CD playback
  • Rip audio CD's to MP3 format
  • Automatic audio CD metadata download

Please note this project is in the process of migrating from SourceForge, a small wiki is available there: https://sourceforge.net/p/dukebox/wiki/Home/
